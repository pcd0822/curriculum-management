Google Apps Script API 설정

배포된 Google Apps Script 웹 앱 URL을 입력하세요. 이 URL은 데이터베이스(Google Sheets)와의 통신에 사용됩니다.

진로 추천 · 커리어넷 (Netlify)

학생 화면의 AI + 커리어넷 추천은 정적 호스팅이 아닌 Netlify 등에 배포할 때 Functions로 동작합니다. OpenAI 연동은 환경변수 OPENAI_API_KEY를 설정하세요.

커리어넷 실제 API를 쓰려면 Netlify에 다음을 설정합니다(없으면 진로 키워드 기반 샘플 과목이 표시됩니다).

  • CAREERNET_API_BASE_URL — API 호스트(예: https://api.example.go.kr)
  • CAREERNET_API_KEY — 발급 키
  • CAREERNET_API_PATH — 선택, 기본 /recommend

관리자 웹은 학교 시트·Apps Script와 통신하고, AI·커리어넷 호출만 Netlify Functions를 경유합니다. API 키는 클라이언트에 넣지 마세요. 커리어넷 API의 정확한 URL·요청/응답 형식은 확정되는 대로 netlify/functions/careernet-recommendation.js에 맞춰 수정하면 됩니다.
